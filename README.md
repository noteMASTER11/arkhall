# 🏙 АРКХОЛЛ — Конструктор городской планировки

> Инструмент для визуального проектирования городов в сеттинге Metalismatic.  
> Браузерное приложение, не требует установки — открывай и строй.

---

## ✨ Возможности

- **Сетка до 40×40 кварталов** — настраивается под любой масштаб города
- **20+ типов объектов** — муниципальные здания, жильё, инфраструктура, ландшафт
- **Умное объединение** — дороги, стены, площади и ландшафт сливаются в единые зоны при соприкосновении
- **Текстуры** — у каждого типа своя визуальная подача: брусчатка, волны, горы, кирпичная кладка, трава
- **Фон карты** — одним кликом покрыть все пустые клетки водой, землёй или горами
- **Свои здания** — добавляй собственные типы с выбором цвета и размера
- **Экспорт в PNG** — красивая карта с рамкой, заголовком, легендой и датой
- **Сохранение / загрузка** — формат JSON, можно продолжить работу позже

## 🖱 Управление

| Действие | Как |
|---|---|
| Поставить здание | Перетащить из панели на карту |
| Выбрать | Клик по зданию |
| Переместить | Зажать и тянуть выбранное |
| Изменить размер | Тянуть за угловой маркер |
| Удалить | ПКМ по зданию или клавиша `Del` |
| Снять выделение | `Escape` |

---

## 💾 Сохранение и экспорт

- **Сохранить** — скачивает файл `.json` с текущей планировкой
- **Загрузить** — открывает сохранённый `.json`
- **PNG** — экспортирует карту как изображение с заголовком, легендой и подписью

---

## ⚙️ Свои здания

Кнопка **＋ Своё** открывает диалог:
- Название
- Размер (в клетках)
- Цвет фона, рамки, текста (с палитрой пресетов)

Свои типы сохраняются вместе с планировкой в JSON.

---

## 🗺 Фон карты

В нижней части панели — кнопки **Вода / Земля / Гора**.  
Нажми любую — все пустые клетки заполнятся выбранным ландшафтом.  
Повторный клик или кнопка **Сброс** убирают фон.

---

## 🚀 Запуск

Просто открой файл `index.html` в браузере — или зайди по ссылке если проект опубликован.  
Интернет нужен только для загрузки шрифтов (Cinzel, IM Fell English).
